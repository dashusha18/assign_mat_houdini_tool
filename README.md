# Assigning materials Tool For Houdini

TO INSTALL:

Step 1. Unpack the Tool:
Unzip and locate the "assign_materials_USD_tool" to any location on your computer.

Step 2. Update the UI File Path:
Open the .py file and locate the following line:

#Path to my ui
    ui_path = r'P:\all_work\userNames\ds24abc\Python_tool\UI_Tool_window.ui'

Replace the path with the correct location of the .ui file on your machine.
Make sure to keep the filename UI_Tool_window.ui unchanged.

Step 3. Create a Houdini Shelf Tool:
In Houdini, create a new tool on the shelf. Paste the entire content of the .py file into the script section.

After pressing "Accept" you should have a QT window appears with further instructions for using the tool.
